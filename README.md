# omakasetli

グループ全体の好みを考慮した楽曲リスト自動生成ツール

## サンプルプログラムの実行方法

1. API トークンの設定

```bash
cd backend
echo FIXSTARS_API_TOKEN={{APIトークン}} > .env
```

2. 依存パッケージのインストール

```bash
poetry install
# or
pip install -r requirements.txt
```

3. サンプルプログラム

`backend/sample.ipynb` でサンプルデータを用いて実行可能です。
